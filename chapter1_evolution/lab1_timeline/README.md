# Lab 1: DevOps Timeline

## Objective
Understand the historical evolution of software delivery models: Waterfall → Agile → DevOps.

## Activity
- Draw a timeline showing key milestones in software engineering.
- Mark when Agile practices emerged and when DevOps gained traction.
- Compare how each model handles speed, collaboration, and feedback.

## Reflection Questions
- What limitations did the Waterfall model introduce?
- How did Agile improve iteration speed?
- Why was DevOps necessary beyond Agile?

## Case Study Reference
- Amazon’s shift from monolithic releases to continuous delivery.
- Netflix’s adoption of DevOps for streaming scalability.

## Notes
This is a conceptual exercise — no technical setup required.
