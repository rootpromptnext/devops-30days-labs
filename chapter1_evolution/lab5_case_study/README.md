# Lab 5: Case Study Reflection

## Objective
Analyze real‑world DevOps case studies and extract lessons.

## Activity
- Read a case study (Amazon, Netflix, Spotify).
- Summarize the challenges faced before DevOps adoption.
- Identify improvements after DevOps practices were introduced.

## Reflection Questions
- What cultural changes were necessary for DevOps success?
- How did tooling support collaboration and automation?
- Which DORA metrics improved most?

## Case Study Reference
- Netflix’s chaos engineering practices.
- Spotify’s squad model for DevOps culture.

## Notes
Conceptual reflection — focus on lessons learned.
