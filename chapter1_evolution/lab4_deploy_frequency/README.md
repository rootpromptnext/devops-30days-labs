# Lab 4: Deployment Frequency

## Objective
Reflect on how deployment frequency impacts stability and innovation.

## Activity
- Compare weekly vs daily vs hourly deployments.
- Discuss trade‑offs between speed and risk.
- Relate deployment frequency to innovation cycles.

## Reflection Questions
- How does frequent deployment reduce batch size of changes?
- Why does smaller batch size lower failure rate?
- How does deployment frequency connect to customer feedback?

## Case Study Reference
- Amazon deploys thousands of times per day, enabling rapid innovation.

## Notes
Conceptual reflection — no technical setup required.
