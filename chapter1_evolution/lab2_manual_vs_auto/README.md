# Lab 2: Manual vs Automated Deployments

## Objective
Compare the risks and benefits of manual vs automated deployments.

## Activity
- Write down steps for a manual deployment (e.g., copying files, restarting servers).
- Write down steps for an automated deployment (e.g., CI/CD pipeline).
- Discuss differences in speed, reliability, and error rates.

## Reflection Questions
- What risks are introduced by manual deployments?
- How does automation reduce human error?
- Which **DORA metrics** are most impacted?

## Case Study Reference
- Etsy’s move from manual deploys to automated pipelines reduced MTTR significantly.

## Notes
Conceptual comparison — no commands required.
